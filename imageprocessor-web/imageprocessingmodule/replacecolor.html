ReplaceColor

Replaces a color within the current image.

Added version 4.1.0

Requests are passed thus:

{% highlight xml %} http://your-image?replace=000|fff&fuzziness=30 http://your-image?replace=[255,0,0,0],[255,255,255,255]&fuzziness=30, http://your-image?replace=black,white&fuzziness=30 {% endhighlight %}
