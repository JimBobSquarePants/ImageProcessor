Filter

Applies a filter to the current image.

Available filters are:

  • blackwhite
  • comic
  • gotham
  • greyscale
  • hisatch
  • invert
  • lomograph
  • losatch
  • polaroid
  • sepia

Requests are passed thus:

{% highlight xml %} http://your-image?filter=comic {% endhighlight %}
